# Portable Solar Phone Charger

## Overview
This project demonstrates the design and implementation of a portable solar-powered phone charger.

## Components Used
- Solar panel
- Buck converter
- USB output module

## Working Principle
The solar panel produces a 12v output that is then stepped down by a buck converter which gives a constant 5v supply energy used to charge mobile devices.

## Applications
- Emergency charging
- Outdoor activities
- Rural areas
